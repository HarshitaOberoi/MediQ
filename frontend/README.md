## 🚀 Features

- **Dynamic Dashboards:** Tailored interfaces for Patients, Doctors, Chemists, and Hospital Admins.
- **Real-time Search:** Symptom-based search mapping to specialized doctors.
- **Interactive Booking:** Live token generation and appointment scheduling.
- **Consultation Hub:** Digital prescription management and patient history viewing.
- **Responsive UI:** Built with Tailwind CSS and Framer Motion for a fluid experience.

## 🛠️ Tech Stack

- **Framework:** React 19 (Vite)
- **State Management:** Zustand
- **Form Handling:** React Hook Form + Zod
- **Routing:** React Router 7
- **Styling:** Tailwind CSS 4
- **Icons:** Lucide React
- **Notifications:** React Toastify
- **3D Elements:** Three.js + React Three Fiber (for background effects)

## 📁 Structure

- `src/api`: Axios instances and API services.
- `src/components`: Reusable UI elements (Buttons, Inputs, Navbars).
- `src/pages`: Application views (Dashboards, Registration, Search).
- `src/store`: Zustand state stores.
- `src/hooks`: Custom React hooks (e.g., WebSockets).
- `src/schemas`: Zod validation schemas.

## ⚙️ Development

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run Development Server:**
   ```bash
   npm run dev
   ```

3. **Build for Production:**
   ```bash
   npm run build
   ```
